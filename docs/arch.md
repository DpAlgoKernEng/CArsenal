# 目录架构设计

## 📋 **项目总览**

**项目名称**: CArsenal (C/C++ Core Arsenal Collections)
**项目类型**: C/C++ 学习资源与实践项目综合仓库
**核心理念**: 建立一个集学习、实践、参考于一体的C/C++开发工具箱

## 🎯 **项目本质**

**这是一个精心设计的C/C++学习与实践综合平台，这不是一个单一应用，而是一个C/C++开发者的"军火库"**：

- **学习参考**：完整的C/C++语言特性文档和跨语言对比
- **实践项目**：60+个分级项目（初级→中级→高级）
- **工具链示例**：现代C++开发环境完整配置
- **模块化库**：可复用的功能组件（如colorfmt彩色输出）

## 🔍 **关键洞察**

### **1. 目录结构解析**

```bash
CArsenal/
├── src/                   # 主程序源码
│   └── CArsenal.cpp       # 演示程序（展示Boost库使用）
├── library/               # 项目自有库模块
│   ├── colorfmt/          # 彩色输出库（ANSI转义码）
│   ├── cmdline_parser/    # 命令行参数解析库
│   └── library.md         # 库模块说明
├── program/               # 实践程序集合（60+个项目）
│   ├── calculator_qt/     # 计算器程序（C/C++实现）
│   ├── editor_tiny/       # 轻量级类vim编辑器
│   └── program.md         # 详细项目列表
├── include/               # 公共头文件接口
├── third/                 # 第三方依赖管理
│   ├── log_format/       # 日志库（log.c, spdlog, fmt）
│   ├── cmdline_parser/   # 命令行解析库（gflags, cli11）
│   ├── lib_manager/      # 包管理器（vcpkg）
│   └── third.md          # 依赖管理文档
├── test/                  # 测试框架
│   ├── benchmark/        # 性能测试
│   ├── example/          # 示例测试
│   └── integration/      # 集成测试
├── reference/            # 全面的学习参考资料
│   ├── c/               # C语言参考
│   ├── cpp/             # C++语言参考（完整语法特性）
│   ├── rust/            # Rust参考
│   ├── go/              # Go参考
│   └── book.md          # 计算机书籍推荐
├── docs/                 # 项目文档
├── build_cmake/          # CMake构建目录
└── vcpkg_installed/      # vcpkg依赖安装目录
```

### **2. 架构设计专业**

- **模块化分离**：library（库）/program（程序）/third（三方库）清晰隔离
- **现代工具链**：vcpkg + CMake 3.22 + Clang工具套件
- **完整测试**：基准测试、示例测试、集成测试三层架构

## 💻 **技术栈与工具链**

### **编程语言**

- **C++17**（主语言，CMake配置）
- **C11**（支持，CMake配置）
- **Assembly**（支持，CMake配置）

### **构建系统**

- **CMake 3.22+**（现代CMake实践）
- **vcpkg**（跨平台包管理器）
- **多种构建类型支持**：Debug, Release, RelWithDebInfo, MinSizeRel

### **核心依赖（通过vcpkg管理）**

- **Boost 1.90.0**（全面依赖）：
  - `boost-program-options`：命令行参数解析
  - `boost-log`：日志系统
  - `boost-thread`：多线程支持
  - `boost-filesystem`：文件系统操作
  - `boost-system`：系统支持
  - `boost-date-time`：日期时间处理
  - `boost-chrono`：时间功能

### **开发工具链**

- **Clang工具套件**：
  - `.clang-format`：基于LLVM风格的自定义代码格式化
  - `.clang-tidy`：全面的静态代码分析配置
  - `.clangd`：语言服务器配置
- **VS Code集成**：完整的开发环境配置
- **Git子模块**：管理第三方库源码

## 🎯 **核心功能模块**

### **1. 库模块（library/）**

- **colorfmt**：轻量级彩色终端输出库
  - 支持6种颜色（紫、蓝、绿、黄、红、粉）
  - 基于ANSI转义码实现
  - 简单的API设计：`print_color()`函数族

### **2. 实践项目（program/）**

项目列表包含**60+个**分层次项目：

- **初级项目**（9个）：命令行工具、计算器、文件操作等
- **能力提升项目**（11个）：定时器、网络库、STL实现等
- **中级项目**（5个）：LRU缓存、HTTP服务器、JSON解析器
- **实战强化项目**（15+个）：线程池、智能指针、数据库等
- **高级项目**（20+个）：编译器、操作系统内核、分布式系统

### **3. 学习参考（reference/）**

- **C语言**：完整语法、标准库、最佳实践
- **C++**：现代C++特性、模板、并发编程
- **多语言对比**：C/C++/Rust/Go特性对比
- **书籍推荐**：计算机科学经典书籍清单

### **4. 示例程序**

- **CArsenal主程序**：展示Boost库综合使用
- **calculator_qt**：多实现计算器（递归下降、逆波兰等算法）
- **editor_tiny**：轻量级类vim编辑器
